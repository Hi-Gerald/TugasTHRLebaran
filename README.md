## Deskripsi Singkat
Aplikasi CampusInfo adalah platform informasi mahasiswa yang menyediakan daftar pengumuman kampus, jadwal kuliah harian, dan manajemen profil mandiri.

## Fitur Utama
1. **Pengumuman**: Daftar berita kampus menggunakan RecyclerView dan ListAdapter.
2. **Detail Berita**: Navigasi menggunakan Safe Args untuk mengirim data ID.
3. **Jadwal Kuliah**: Menampilkan list jadwal kuliah secara rapi.
4. **Profil & SharedPrefs**: Fitur edit nama yang tersimpan permanen di perangkat.
5. **Dark Mode**: Dukungan tema gelap baik secara manual maupun mengikuti sistem.
